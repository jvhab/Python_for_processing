В OpenStreetMap XML встречаются теги node, которые соответствуют некоторым точкам на карте.
Ноды могут не только обозначать какой-то точечный объект, но и входить в состав way (некоторой линии, возможно замкнутой) и не иметь собственных тегов.
Для доступного по ссылке https://online.hse.ru/pluginfile.php/604906/question/questiontext/805598/1/789299/task_8_2_3_map2.osm фрагмента карты посчитайте, сколько всего тегов node встречается на этой карте.
В качестве ответа введите одно число.
